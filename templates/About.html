      Features of Encryption:
              
                        1) SHA256 Hashing algoritham

                        2) One-Way Encryption

                        3) Decryption key 

                        4) Key is 128 bit

                        5) Randomly generated Key 
              
              
              Info About the "KEY" :
                          The key is generated Randomly and the Hashed value can only be Dehashed with that same KEY
                          If the KEY is by any chance gets lost There is no way possible to get the Original value back.

                          This is because the hash of a string changed each time it is HASHED, even with a same KEY
                          Thus, leaving no chance to get a HASH from a RANDOM string which is same as the Original HASH.
